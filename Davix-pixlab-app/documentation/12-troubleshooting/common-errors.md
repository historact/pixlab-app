# Common Errors

- `invalid_api_key`/`key_expired`: Check API key value and validity window.
- `missing_field`: Ensure required body parameters are provided per endpoint.
- `monthly_quota_exceeded`: Customer keys exceeded monthly file limit.
- `rate_limit_exceeded`: Public IP-based daily cap hit.
- `payload_too_large`/`too_many_files`: Reduce upload size or file count.
